#### /boot (500MiB)



<aside>
💡

- /boot는 커널/부트로더 관련 파일이 들어가는 영역이다.
- 커널 업데이트가 여러 번 누적되면 용량이 부족해질 수 있으니, 운영에서는 기준 용량을 더 크게 잡기도 한다.
</aside>

#### 3-3-2) swap (8192MiB)

![화면 캡처 2026-04-07 104721.jpg](%EB%A6%AC%EB%88%85%EC%8A%A4%208%209%20%EC%84%A4%EC%B9%98/%ED%99%94%EB%A9%B4_%EC%BA%A1%EC%B2%98_2026-04-07_104721.jpg)

<aside>
💡

- Oracle 설치 사전 점검에서 swap 관련 권고사항이 걸리는 경우가 많다.

</aside>

### 5. 네트워크 설정

호스트 이름을 rac1으로 설정하고, enp0s3, enp0s8 둘 다 켠다.

![image.png](%EB%A6%AC%EB%88%85%EC%8A%A4%208%209%20%EC%84%A4%EC%B9%98/image%2010.png)

#### DNS 서버 주소 확인

먼저 Windows 명령 프롬프트를 열고 `ipconfig /all`로 DNS 서버 주소를 확인한다.

![image.png](%EB%A6%AC%EB%88%85%EC%8A%A4%208%209%20%EC%84%A4%EC%B9%98/image%2011.png)

#### Public IP 설정(enp0s3)

설정으로 들어가 enp0s3(VirtualBox의 어댑터1)을 다음과 같이 public IP로 설정한다.

![image.png](%EB%A6%AC%EB%88%85%EC%8A%A4%208%209%20%EC%84%A4%EC%B9%98/image%2012.png)

IPv4를 수동으로 설정한다. 서브넷 마스크 255.255.255.0은 24로 입력해도 같은 값이다.

DNS servers는 위에서 확인한 주소로 입력한다.

![image.png](%EB%A6%AC%EB%88%85%EC%8A%A4%208%209%20%EC%84%A4%EC%B9%98/image%2013.png)

#### Private IP 설정(enp0s8)

enp0s8(VirtualBox의 어댑터2)도 나머지는 동일하게 private IP로 설정한다.

IPv6는 설정하지 않고 저장한다.

![image.png](%EB%A6%AC%EB%88%85%EC%8A%A4%208%209%20%EC%84%A4%EC%B9%98/image%2014.png)
