# Oracle 19c RAC Installation Guide

Oracle 19c RAC(Real Application Clusters) 환경 구축 가이드입니다. 

---

## 환경 구성

| 항목 | 내용 |
|------|------|
| OS | Oracle Enterprise Linux 8.4 |
| DB 버전 | Oracle Database 19.3 |
| Grid 버전 | Oracle Grid Infrastructure 19.3 |
| 가상화 | VirtualBox 7.0 |
| 노드 수 | 2 Node RAC |
| 메모리 | 각 노드 8GB |
